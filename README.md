# Secure Application Programming Project

## Project Description
This project demonstrates the difference between insecure and secure coding practices in a web application.

The application is a simple blog system that allows users to register, log in, create posts and comment on other posts.

## Technologies Used
- Node.js
- Express
- SQLite3
- HTML/CSS
- JavaScript

## Branch Structure
- main - stable version
- insecure - version with intentional vulnerabilities
- secure - version with security protections applied

## How to Run the Application

### Prerequisites
- Node.js installed
- npm installed

### Installation
1. Clone the repository:
   git clone <your-repo-url>

2. Install dependencies:
   npm install

3. Run the application:
   node server.js

More detailed setup will be provided per branch.
